# quantum-Image-encryption-and-decryption-
A lightweight, 4-qubit quantum secure communication framework designed for NISQ environments, integrating Hadamard-based positioning, XOR shared-key encoding, and Toffoli gate embedding for secure, blind hidden-information extraction. To minimize qubit resource dependency and circuit complexity, a 2X2 grayscale image model is utilized, mapping its four spatial coordinates to a compact 2-qubit positional system. This minimal representation maintains a shallow circuit depth that bypasses severe near-term hardware limitations. Ultimately, the 2X2 layout serves as a practically feasible baseline to experimentally validate the protocol's blind extraction and shared-key recovery capabilities within realistic quantum simulation constraints.
